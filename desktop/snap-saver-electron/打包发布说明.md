# Electron 版打包发布说明

目标：打成 Windows 安装程序，用户电脑不需要安装 Node.js、npm、Python。

## 一、打包命令

在 `desktop/snap-saver-electron/electron` 目录执行：

```bat
npm run dist
```

这个命令会自动做两件事：

1. 用 PyInstaller 把 `backend.py` 打成 `dist-backend/backend.exe`
2. 用 electron-builder 把 Electron 程序打成安装包，并把 `backend.exe` 内置进去

## 二、输出文件

安装包位置：

```text
desktop/snap-saver-electron/electron/release/智能截图软件 Setup 2.0.0.exe
```

把这个 exe 发给用户即可。用户安装后可以直接运行，不需要 npm、Node.js、Python。

## 三、打包机需要的环境

只有打包这台电脑需要：

- Node.js / npm
- Python
- Python 依赖：`pip install -r requirements.txt`
- Electron 依赖：`cd electron && npm install`

普通用户电脑不需要这些环境。

## 四、关键文件

- `electron/package.json`：Electron Builder 配置和打包命令
- `snap-saver-backend.spec`：PyInstaller 后端打包配置
- `electron/main.js`：开发期启动 `backend.py`，安装包内启动内置 `backend.exe`
