# 换电脑后怎么继续干活

GitHub 上只有**源码**（node_modules、Electron 运行时、依赖都没传，太大）。
新电脑拉下来后，要先把环境装好，才能 `npm start`。

## 一、先决条件（新电脑装这两个）

1. **Node.js**（v18 以上）— https://nodejs.org 下载安装
2. **Python**（3.10 以上）— https://python.org 下载安装，安装时勾选 "Add to PATH"

装完在 cmd 里验证：
```
node --version
python --version
```

## 二、拉代码

```
git clone https://github.com/likailongliyanni/smallsoft.git
cd smallsoft/desktop/snap-saver-electron
```

## 三、装 Python 依赖

```
pip install -r requirements.txt
```
（就 Pillow + openpyxl，其它都是 Python 标准库）

## 四、装 Electron（关键：走国内镜像，否则运行时下不动）

```
cd electron
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install --registry=https://registry.npmmirror.com
```

> 这一步会下载约 270MB 的 Electron 运行时。不走国内镜像的话会卡在国外服务器下不下来，
> 表现为「装完了但 npm start 报 Electron failed to install」。

## 五、启动

```
npm start
```
（在 electron 目录下）

## 六、以后每天干活

代码改完提交：
```
cd smallsoft
git add -A
git commit -m "改了啥"
git push
```

回到原电脑或换电脑，先拉最新：
```
git pull
```

---

## 提醒：截图功能依赖 Windows

`CtrlDragHook`（全局 Ctrl 拖框截图）用的是 Windows API（ctypes 调 user32），
**只能在 Windows 上跑**。Mac/Linux 上界面能开，但截图功能用不了。
