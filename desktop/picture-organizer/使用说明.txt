图片整理器 使用说明（开网页 + 自动归档版）

软件定位：
你在浏览器里自己找到真正的原图「另存为」，软件负责自动开网页、把存下来的图自动搬到对应商品文件夹并改名。

基本流程：
1. 打开 图片整理器.exe。
2. 准备 Excel/CSV/TXT 模板：
   A列：名称，也就是保存图片用的文件夹名。
   B列：链接，比如京东、天猫商品链接。
3. 点击「导入文件」导入模板。
   也可以复制 AB 两列数据后点击「从剪贴板导入」。
4. 选择输出目录。
5. 确认监控目录（默认是系统「下载」文件夹，浏览器另存图片时保存到这里即可；可改）。
6. 「主图/详情数量」只是进度提示，存多存少都不会自动跳行。
7. 点击「开启整理 F9」或按 F9：主窗口隐藏，置顶小浮窗出现，
   软件用你系统默认的浏览器打开当前行链接（登录状态、插件都是现成的）。
8. 在网页里找到真正的原图，右键「图片另存为」，保存到监控目录。
   软件监控到新图片后自动搬走并改名：
   输出目录\商品名\主图\pic1.jpg（统一转 JPG，透明背景自动补白）。
9. 主图存完，点浮窗上的「详情」，之后保存的图归入 详情\。
10. 点「下一行 ▶」：切到下一行、自动打开下一条链接、类别自动回到「主图」。
    当前商品一张图都没归档时点下一行，会弹确认提醒，防止手滑。
11. 重复，直到列表做完。点浮窗「结束整理」可随时退出工作模式。

浮窗按钮说明：
- 主图 / 详情：切换之后保存的图片归到哪个文件夹。
- 下一行 ▶：手动进入下一行（0 张图时会确认）。
- 跳过当前：这一行不要了，直接切到下一行。
- 复制上一个：当前商品图片和上一行完全一样时使用，
  会把上一行的「主图/详情」完整复制过来，然后自动切到下一行。
  适合 SKU 规格不同但图片相同的场景，比如 250ml 和 500ml。
- 撤销上一张：万一下载目录混进不相干的图被误归档，一键删除并退回编号。
- 显示程序 F9：调出主窗口看列表进度。
- 结束整理：退出工作模式。

监控规则（重要）：
- 只认「开启整理之后」新出现的文件，监控目录里原来的文件一律不动。
- 浏览器下载中的临时文件（.crdownload/.tmp/.part）会自动忽略，等下载完成才归档。
- 只处理图片格式（jpg/jpeg/png/webp/gif/bmp/avif），其他文件不碰。
- 连续快速保存多张时，按文件出现顺序依次编号。
- 归档成功后，监控目录里的原始下载文件会被删除（已经搬到输出目录了）。

保存规则：
- 不管原图是什么格式（webp/png/...）统一转成 JPG，透明背景补白。
- 文件名 = 前缀 + 序号（默认 pic1.jpg、pic2.jpg...），主图和详情各自独立编号。
- 「覆盖同名文件」勾选时同名直接覆盖；不勾选时自动顺延编号。

截图去水印（推荐流程）：
1. 主界面工具栏点「截图去水印」打开窗口，填入阿里云百炼 API Key。
2. 在浏览器里浏览商品图，按 F8（或点窗口里的「截图 (F8)」按钮）：
   屏幕变暗后，按住鼠标左键拖框选要截的区域，松开自动保存，不需要另存为。
   想截几张截几张，列表里会累计。按 Esc 可取消当前这次框选。
3. 攒够后点「一键去水印」：
   - AI 先逐张检测有没有水印，结果显示在「水印检测」列（有水印/无水印/检测失败）。
   - 然后弹出确认框：含水印的默认勾选，可以取消不想处理的，也可以补勾漏判的。
   - 确认框实时显示费用：已选 N 张 × 0.10 元 = X.XX 元。
   - 再次确认后才真正开始去水印，按 0.10 元/张计费。
4. 结果保存到「保存目录\去水印\」，原始截图在「保存目录\截图\」，原图不动，统一 JPG。
5. 说明：检测会用少量 AI（每张约几厘）；去水印是 0.10 元/张。
   只处理你自己拥有或已获授权的图片。

AI 批量去水印（处理已有的图片文件夹）：
1. 主界面工具栏点「AI 去水印」打开窗口。
2. 填入阿里云百炼的 API Key（和网站后台同一个 DASHSCOPE_API_KEY）。
   第一次填写后保存在软件目录的「图片整理器配置.json」，下次自动带出。
3. 「选择文件夹」批量处理整个目录（可勾选包含子文件夹），
   或「选择图片」一次多选几张图。
4. 点「开始去水印」，结果保存位置：
   - 文件夹模式：同级的「原文件夹名_去水印」目录，保持原目录结构。
   - 选图模式：图片所在目录的「去水印」子文件夹。
   - 原图一律不动，结果统一转成 JPG。
5. 处理规则：
   - 小于 512px 的图自动放大到接口要求的尺寸，大于 4096px 自动缩小。
   - 超长图（长宽比超过 8:1，比如详情长图）接口不支持，会提示跳过。
   - 每张图大约几秒到几十秒，按张计费（阿里云百炼 wanx2.1-imageedit）。
   - 失败自动重试一次；API Key 无效或欠费时自动停止整批。
   - 处理过的「去水印」目录再次批量时会自动跳过，不会重复花钱。
6. 重要：请只处理你自己拥有或已获得授权的图片。

常见提示：
- F9 是全局快捷键，工作模式中再按 F9 可以显示/隐藏主窗口。
- 如果 F9 被其他软件占用，可以点击程序里的「开启整理 F9」按钮。
- 「切行后自动打开链接」默认开启；不需要自动打开时可以取消勾选。
- 软件只负责打开链接，不会自动关闭浏览器窗口或标签页。
- 输出目录里会自动创建「文件夹名\主图」和「文件夹名\详情」。
- 文件夹名中的 Windows 非法字符会自动替换为下划线。
